# Elder Assist 👴👵

## 📌 Description

Elder Assist is a simple web-based project designed to support elderly people by providing easy access to essential services, guides, and user-friendly features like sign-in and sign-up pages.

## 🎓 Academic Note

This project was created as part of academic coursework.

## 🚀 Features

* User-friendly interface for elderly users
* Sign In and Sign Up functionality
* Service pages with helpful information
* Organized guides and service sections
* Clean and simple design

## 🛠️ Technologies Used

* HTML5
* CSS3
